# WebRtcExample

透過Socket.io作為Signaling Server協助RTCPeerConnection做訊息交換，達成簡單的WebRTC聊天室

## Getting Started

如何開始使用指南。

### Installing

```bash
npm install
```

## Running the tests

```bash
npm run dev
```

## Deployment

```bash
npm run start
```
